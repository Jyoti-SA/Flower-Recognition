# Flower-Recognition
Image recognition tool for flower clasification.

The dataset used is available [here](http://www.robots.ox.ac.uk/~vgg/data/flowers/17/)

There are 17 differents class of flowers on this dataset. Each class has 80 images to work with.

Classes

* **Daffodil**: 1-80
* **Snowdrop**: 81-160
* **LillyValley**: 161-240
* **Bluebell**: 241-320
* **Crocus**: 321-400
* **Iris**: 401-480
* **Tigerlily**: 481-560
* **Tulip**: 561-640
* **Fritillary**: 641-720
* **Sunflower**: 721-800
* **Daisy**: 801-880
* **Colt's Foot**: 881-960
* **Dandelion**: 961-1040
* **Cowslip**: 1041-1120
* **Buttercup**: 1121-1200
* **Windflower**: 1201-1280
* **Pansy**: 1281-1360
